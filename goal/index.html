你想成为什么样的人

自我定位： 一个懂产品，能够项目管理的架构师。全面发展，以架构师为主，产品和项目管理为辅。

分析

  • 架构师
    • Linux
    • Apache/ Nignx
    • MySQL 学习网易MySQL DBA课程
    • Git 项目流程
    • PHP Drupal8/ Laravel， 两者都基于Symfony.
  • 项目管理
    • SCRUM敏捷式开发 学习敏捷式开发和项目管理的思想， 将项目管理的思想用到自己的生活当中
    • 英语
      • 听 在电话或面对面的交流中，能够听懂理解对方的意思
      • 说 能够表达清楚自己的意思，对待程序员和客户采用不同的交流技巧
      • 读 能够快速阅读英语资料，归纳大意。大多数单词都认识
      • 写 能够再技术社区中清楚的表达自己所要阐述的内容；邮件。
  • 产品
    • 学习 网易云课堂的 产品经理课程。

时间

2015年 10月 16日 到 2016年10月16日。

方法

  1. 坚持不懈的写博客
  2. 看视频 + 看书 + 实践。 接触新技术最快的方法是像已深入掌握这项技术的人学习，然后不断重复。简单来说就是两点。一个是学，另一个是练。
  3. 总结消化后，分享自己的观点能够使你更加进步。
